# Image Editor

Multi-layer compositor and single-image crop tool built with React + Vite.

Live site: https://vikramsg.github.io/Experiments/

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run test`
- `npm run e2e`

## Notes

- The app is deployed via GitHub Pages from `/client_tools/image-editor`.
- Vite base path defaults to `/Experiments/` in production.
